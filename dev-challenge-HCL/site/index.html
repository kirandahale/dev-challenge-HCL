Updating Price Table

Approach for Implementation

  1. Subscribed to /fx/prices to get data for the table.
  2. Created a fucntion createTable to create table dynamically and update data on DOM when data is getting updated.
  3. Called createTable function on table data is updated
  4. Created sparkline data as per calculation mentioned in assignment, draw the sparkline
  5. Added 30 sec time interval to update sparkline data, so we can show last 30 sec best mid price data.
  6. Sorted table by Last changed ask price amount column in descending order by javascript sort() method.

Frameworks/Liabraries would like to use:

React JS I would like to use for this solution, because virtual DOM by which we can update only the changed data and for better performance for live update table. Bootstrap for table, easy to paginate and manage data at client side. In this case, would like to add server side pagination to limit data to push on client side so that DOM would not break after its limit exceeds.
